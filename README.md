# WebPaint

An open-source web-based painting application that allows users to create digital artwork directly in their browser. WebPaint provides a simple, intuitive interface with powerful drawing tools.

## ✨ Features

- 🎨 Multiple brush types and sizes
- 🌈 Full color palette with opacity control
- 🗑️ Eraser and clear canvas functionality
- 💾 Save artwork as PNG/JPEG
- ↩️ Undo/Redo functionality
- 📱 Responsive design for desktop and mobile devices

## 🖼️ Try

Visit [WebPaint](https://webpainty.vercel.app/) to try the application online!

## 🚀 Installation

To run WebPaint locally:

```bash
# Clone the repository
git clone https://github.com/awmie/webpaint.git

# Navigate to project directory
cd webpaint

# Install dependencies
npm install

# Start development server
npm run dev
```

Then open your browser and navigate to `http://localhost:5173` (or the port shown in your terminal).

## 🔧 Usage

1. Select your desired tool from the toolbar
2. Choose a color from the color picker
3. Adjust brush size and opacity as needed
4. Draw on the canvas
5. Save your artwork using the download button
